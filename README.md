# atportalen
SU AT Portalens APP

# Vi använder Kivy för att skapa appen.
Finns här https://kivy.org/#home

# Appen ska ha

1. Inloggnings-sida.
2. Sida med
  * Anställning - Ledighet
  * Tjänstgöring - Schema - Legitimation
  * Utbildning - Kurser
  * Forskning - Förbättring
  * Handledning
  * AT-råd - AT-rum
  * Kontakt
  * Regionkalendern
3. Sida som kan presentera texterna från AT-portalen
